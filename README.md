# Nest.js API

## Description

This is a REST API that I use for my personal portfolio. This API got a few endpoints that return all my experience and education so a Frontend can use it.
Also added security with a login/JWT to the create endpoint.

### :computer: Technologies used

- Nest.JS
- Typescript
- SQLite
- JWT

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
